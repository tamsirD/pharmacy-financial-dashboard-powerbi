#  Pharmacy Financial Dashboard – Power BI Project

##  Project Overview

This project presents an interactive Power BI dashboard designed for a pilot pharmacy in order to monitor:

- Financial performance
- Sales activity
- Inventory management
- Supplier purchases
- Product profitability
- Operating expenses

The objective is to provide a dynamic decision-support tool for pharmacy management through automated KPIs and business-oriented analytics.

---

#  Business Objectives

✅ Monitor pharmacy revenues and expenses  
✅ Analyze financial performance  
✅ Optimize stock management  
✅ Improve purchasing decisions  
✅ Detect slow-moving and high-risk products  
✅ Support operational and financial decision-making

---

#  Tools & Technologies

- Power BI
- Power Query
- DAX
- Excel
- Data Modeling (Star Schema)

---

# 📊 Dashboard Structure

---

# Page 1 — Financial Performance Overview

This page provides a global financial view of the pharmacy activity.

## Main KPIs
- Total Revenue
- Purchases TTC
- Operating Expenses
- Estimated Result
- Purchases / Revenue Ratio
- Expenses / Revenue Ratio

## Visualizations
- Expense Distribution
- Financial Structure Analysis
- Daily Sales & Purchases Monitoring
- Dynamic Financial Commentary

## Features
✅ Dynamic monthly filtering  
✅ Automated business commentary  
✅ Financial KPI monitoring  
✅ Interactive visuals

---

#  Page 2 — Stock & Product Analysis

This page focuses on stock monitoring and product performance.

## Main KPIs
- Number of References
- Quantity Sold
- Quantity Purchased
- Total Stock
- Purchase Stock Value
- Sales Stock Value

## Visualizations
- Top Products by Stock Value
- Best-Selling Products
- Stock vs Sales Analysis
- Product-Level Inventory Monitoring

## Business Value
✅ Inventory optimization  
✅ Stock rotation monitoring  
✅ Identification of strategic products  
✅ Prevention of stock shortages

---

# Page 3 — Supplier & Profitability Analysis

This page analyzes supplier purchases and product profitability.

## Main KPIs
- Number of Suppliers
- Number of Delivery Notes
- Dormant Products

## Visualizations
- Top Suppliers by Purchase Amount
- Products with Purchases Higher Than Sales
- Purchase vs Sales Gap Analysis
- Business Insights Commentary

## Business Value
✅ Supplier performance monitoring  
✅ Expense control  
✅ Detection of overstocked products  
✅ Improvement of purchasing strategy
